# Project Overview

Habit Quest Analytics is a Streamlit dashboard for managing habits and to-do items as RPG-style quests.

The initial product focuses on simple quest tracking, XP rewards, character levels, and habit consistency analytics.

## Goals

- Represent habits and tasks as quests.
- Award XP when quests are completed.
- Calculate player level from total XP.
- Show consistency and completion metrics.
- Keep business logic testable outside Streamlit pages.

## Non-Goals

- Authentication.
- Machine learning.
- External API integrations.
- Multiplayer or social features.
